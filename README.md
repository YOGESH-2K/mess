Follow these steps to install and run the Mess Solution:
  
 Navigate to the project directory:

   ```
   cd MessSolution
   ```

 Compile the `Run.java` file using `javac`:

   ```
   javac Run.java
   ```

 Run the application by executing the compiled `Run` class:

   ```
   java Run
   ```

 The Mess Solution should now be up and running. Follow the on-screen instructions to interact with the application.
